# LPMC (Local Password Manager) 🔐

**LPMC** — это кроссплатформенный локальный менеджер паролей с графическим интерфейсом, написанный на **C++** и **QML** (фреймворк Qt). Приложение позволяет безопасно хранить ваши учетные записи, защищая их мастер-паролем, а также генерировать новые надежные пароли.

---

## 🌟 Основные возможности

* **Надежное хранение:** Локальное сохранение данных с доступом только через единый мастер-пароль.
* **Система авторизации:** Специальные экраны регистрации (`regPage.qml`) и входа (`admission.qml`) для защиты хранилища.
* **Генератор паролей:** Встроенная утилита (`generatePasswordPage.qml`) для создания сложных, устойчивых к взлому паролей по заданным параметрам.
* **Удобный интерфейс:** Современный, плавный UI, созданный с использованием Qt Quick / QML.
* **Кастомный дизайн:** Собственная строка заголовка (`TitleBar.qml`) и удобное отображение сохраненных данных (`PasswordItem.qml`).

---

## 🛠 Технологический стек

* **Язык программирования:** C++ (стандарт C++11 или выше)
* **Пользовательский интерфейс:** QML / Qt Quick
* **Фреймворк:** Qt 5 / Qt 6
* **Среда разработки:** Microsoft Visual Studio (в проекте используются файлы `.sln` и `.vcxproj`)

---

## 📂 Структура проекта

* `main.cpp` — Точка входа в приложение, инициализация QML движка.
* **C++ Backend:**
  * `PasswordModel.cpp` / `.h` — Класс модели (наследуется от `QAbstractListModel`) для передачи данных о сохраненных паролях из C++ в QML.
  * `FileManager.h` — Логика работы с файловой системой (чтение, запись и, вероятно, шифрование базы паролей).
* **QML Frontend:**
  * `main.qml` — Главное окно приложения.
  * `homePage.qml` — Главная страница с отображением списка всех сохраненных записей.
  * `regPage.qml` — Экран первичной настройки и создания мастер-пароля.
  * `admission.qml` — Экран ввода мастер-пароля при запуске.
  * `generatePasswordPage.qml` — Страница генератора паролей.
  * `PasswordItem.qml` — QML-компонент отдельной карточки пароля.
  * `TitleBar.qml` — Кастомная панель управления окном (заголовок, кнопки закрытия/сворачивания).
